Local Setup Required
This project requires an OpenAI API key for intent detection.
Before running, create a file at: src/main/resources/application.properties
Add the following line with your own API key: openai.api.key=YOUR_KEY_HERE

Do NOT commit your key or the application.properties file.
It is intentionally excluded from version control.
